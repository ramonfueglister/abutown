MATSim Documentation

User Guide

The MATSim user guide (PDF) is an extract and update of the MATSim book.

The user guide chapters contain the most relevant topics for new users.

The user guide is typically updated along with the MATSim class that we teach at TU Berlin every year between April and July.

The Book

The “MATSim Book” is now the authoritative reference.

There is an extract of the first couple of chapters, which contains the most relevant topics for new users. This is also referred to as the “MATSim user guide”.

Some Terminology (i.e. correspondences between MATSim terminology and more standard transport modelling terminology, and reasons why ours is different) can be found here.

Publications

Being an open-source project heavily driven by research organizations world-wide, a lot of publications were written about MATSim. If you are interested in scientific publications about MATSim, the publication lists of the following institutions serve as a good entry point:
