# Active Specs

This directory is intentionally small.

The old historical spec pile was removed from the active spec directory on
2026-06-18 because it contained contradictory product directions and kept
reactivating obsolete fixture constraints.

Current rules:

- Active specs in this directory are the only specs Codex should treat as
  implementation authority.
- `progress.md` is the historical ledger.
- Git history contains the removed historical specs when archaeology is needed.
- Do not add speculative specs here.
- Do not reintroduce fixture names, stale labels, or historical Zurich/OpenGFX
  product threads into active direction.
- Frontend, render, browser-wire, and visual product changes need browser
  screenshot or smoke evidence before they can be called complete.
