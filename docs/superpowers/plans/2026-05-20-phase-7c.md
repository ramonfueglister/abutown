# Phase 7c — Tick-Owned Runtime + Arc-Snapshot Reads Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove `Arc<RwLock<SimulationRuntime>>` from `AppState`. Tick task owns the runtime, mutations flow through an mpsc channel, reads go through `Arc<ArcSwap<RuntimeReadView>>` (lock-free).

**Architecture:** Transitional shim approach — keep `RwLock` working through tasks 1-5 while building the new paths in parallel. Drop `RwLock` only in task 6 when every callsite already uses the new paths.

**Deviation from spec §2:** The spec proposed adding `MobilitySet::Input` and `MobilitySet::Publish` as new bevy SystemSets containing `drain_mutations_system` and `build_read_view_system`. The plan instead does drain + publish in the tick_loop async function (outside the bevy schedule). Functional outcome is identical and the explicit control flow is easier to reason about than relying on bevy's deferred-Commands semantics (which caused the Phase-6 NearStop marker debug). The spec's acceptance criteria are satisfied either way; only the file-level organization differs.

**Tech Stack:** Rust 2021, `bevy_ecs = "0.18"`, `arc-swap = "1"`, `tokio::sync::{mpsc, oneshot}`, `axum`. Backend-only.

**Spec:** `docs/superpowers/specs/2026-05-20-phase-7c-design.md`.

```rust
//! Phase 7c — types for the mutation queue + Arc-Snapshot read view.
//!
//! `Mutation` is the variant flowing through the mpsc channel from any
//! HTTP/WS handler into the tick task. `RuntimeReadView` is the
//! immutable per-tick snapshot published via `ArcSwap`.

use abutown_protocol::{
    ChunkSnapshotDto, MobilityChunkDeltaDto, MobilityChunkSnapshotDto, MobilitySnapshotDto,
    WorldId, WorldSummaryDto,
};
use sim_core::ids::ChunkCoord;
use std::collections::HashMap;

use crate::runtime::{AppliedCommand, CommandRejection, HealthResponse};
use abutown_protocol::ClientCommandDto;
use tokio::sync::oneshot;

/// All mutations to the runtime flow through one channel.
pub enum Mutation {
    ApplyCommand {
        command: ClientCommandDto,
        reply: oneshot::Sender<Result<AppliedCommand, CommandRejection>>,
    },
    SubscriptionDiff {
        added: Vec<ChunkCoord>,
        removed: Vec<ChunkCoord>,
        reply: oneshot::Sender<Vec<MobilityChunkSnapshotDto>>,
    },
    MarkChunkSnapshotsPersisted {
        coords: Vec<ChunkCoord>,
    },
}

/// Lock-free read view of the runtime, published once per tick.
/// Everything readers need is pre-materialized so readers never touch
/// the live World.
#[derive(Clone)]
pub struct RuntimeReadView {
    pub tick: u64,
    pub world_id: WorldId,
    pub mobility_tick: u64,
    pub health: HealthResponse,
    pub world_summary: WorldSummaryDto,
    pub chunk_snapshots: HashMap<ChunkCoord, ChunkSnapshotDto>,
    pub mobility_full_dto: MobilitySnapshotDto,
    pub per_chunk_deltas: Vec<MobilityChunkDeltaDto>,
    pub pulse_sequence: u64,
}
```

In `backend/crates/sim-server/src/app.rs` (just below the `AppState` impl block, before `build_router_from_state`), add a helper that takes `&SimulationRuntime` + a `Vec<MobilityChunkDelta>` and produces a `RuntimeReadView`:

```rust
fn build_read_view_from_runtime(
    runtime: &SimulationRuntime,
    per_chunk: &std::collections::HashMap<
        sim_core::ids::ChunkCoord,
        sim_core::mobility::MobilityChunkDelta,
    >,
    pulse_sequence: u64,
) -> crate::runtime_view::RuntimeReadView {
    let world_id = runtime.world_id_for_persist().clone();
    let mobility_tick = runtime.mobility_tick();
    let mobility = runtime.mobility();

    let per_chunk_deltas: Vec<abutown_protocol::MobilityChunkDeltaDto> = per_chunk
        .iter()
        .map(|(_, delta)| chunk_delta_to_dto(delta, mobility, &world_id, mobility_tick))
        .collect();

    // Pre-materialize per-chunk snapshots for every loaded chunk so HTTP
    // /chunks/{x}/{y} and WS chunk_subscribe init can read without a lock.
    let mut chunk_snapshots: std::collections::HashMap<
        sim_core::ids::ChunkCoord,
        abutown_protocol::ChunkSnapshotDto,
    > = std::collections::HashMap::new();
    for coord_dto in runtime.world_summary().loaded_chunks.iter() {
        let coord = sim_core::ids::ChunkCoord {
            x: coord_dto.x,
            y: coord_dto.y,
        };
        if let Some(snap) = runtime.chunk_snapshot(coord) {
            chunk_snapshots.insert(coord, snap);
        }
    }

    crate::runtime_view::RuntimeReadView {
        tick: mobility_tick,
        world_id: world_id.clone(),
        mobility_tick,
        health: runtime.health(),
        world_summary: runtime.world_summary(),
        chunk_snapshots,
        mobility_full_dto: runtime.mobility_snapshot(),
        per_chunk_deltas,
        pulse_sequence,
    }
}
```

```rust
async fn tick_and_fan_out(state: &AppState) {
    // Phase 1: tick the world under brief write-lock; collect deltas + metadata.
    let (per_chunk, world_id, tick) = {
        let runtime_arc = state.runtime();
        let mut runtime = runtime_arc.write().await;
        let per_chunk = runtime.tick_world_mobility();
        let world_id = runtime.world_id_for_persist().clone();
        let tick = runtime.mobility_tick();
        (per_chunk, world_id, tick)
    }; // write-lock dropped here

    // Phase 2: publish per-chunk deltas into broadcast channels.
    let chunk_channels = state.chunk_channels();
    if !per_chunk.is_empty() {
        let runtime_arc = state.runtime();
        let runtime = runtime_arc.read().await;
        let mobility = runtime.mobility();
        for (chunk, delta) in &per_chunk {
            let Some(sender) = chunk_channels.get(chunk).map(|e| e.clone()) else {
                continue;
            };
            let dto = chunk_delta_to_dto(delta, mobility, &world_id, tick);
            let _ = sender.send(dto);
        }
    }

    // Phase 3: publish the RuntimeReadView (Phase 7c — readers can switch to this).
    {
        let runtime_arc = state.runtime();
        let runtime = runtime_arc.read().await;
        let pulse_sequence = state.view.load().pulse_sequence;
        let view = build_read_view_from_runtime(&runtime, &per_chunk, pulse_sequence);
        state.view.store(Arc::new(view));
    }
}
```

```rust
async fn http_reads_are_lock_free_while_commands_are_in_flight() {
    use std::time::{Duration, Instant};

    let state = AppState::new(SimulationRuntime::new());

    // Spawn 50 background tasks that take the runtime write-lock.
    let mut command_tasks = Vec::new();
    for _ in 0..50 {
        let state = state.clone();
        command_tasks.push(tokio::spawn(async move {
            let runtime = state.runtime();
            let _w = runtime.write().await;
            tokio::time::sleep(Duration::from_millis(10)).await;
        }));
    }

    // 50 concurrent reads of /world via the view path — must not wait.
    let start = Instant::now();
    let mut read_tasks = Vec::new();
    for _ in 0..50 {
        let state = state.clone();
        read_tasks.push(tokio::spawn(async move {
            let _ = state.view.load().world_summary.clone();
        }));
    }
    for t in read_tasks {
        t.await.unwrap();
    }
```

```rust
async fn tick_loop(
    mut runtime: SimulationRuntime,
    mut mutation_rx: tokio::sync::mpsc::UnboundedReceiver<crate::runtime_view::Mutation>,
    view: Arc<arc_swap::ArcSwap<crate::runtime_view::RuntimeReadView>>,
    deltas: broadcast::Sender<ServerMessageDto>,
    chunk_channels: Arc<DashMap<ChunkCoord, broadcast::Sender<MobilityChunkDeltaDto>>>,
    interval: Duration,
) {
    let mut ticker = tokio::time::interval(interval);
    ticker.tick().await;
    loop {
        ticker.tick().await;

        // Phase 0: drain mutations.
        while let Ok(mutation) = mutation_rx.try_recv() {
            apply_mutation_owned(&mut runtime, mutation);
        }

        // Phase 1: tick mobility.
        let per_chunk = runtime.tick_world_mobility();

        // Phase 2: broadcast per-chunk deltas.
        let world_id = runtime.world_id_for_persist().clone();
        let mobility_tick = runtime.mobility_tick();
        for (chunk, delta) in &per_chunk {
            let Some(sender) = chunk_channels.get(chunk).map(|e| e.clone()) else {
                continue;
            };
            let dto = chunk_delta_to_dto(delta, runtime.mobility(), &world_id, mobility_tick);
            let _ = sender.send(dto);
        }

        // Phase 3: build + publish view.
        let prev_pulse = view.load().pulse_sequence;
        let new_view = build_read_view_from_runtime(&runtime, &per_chunk, prev_pulse + 1);
        view.store(Arc::new(new_view));

        // Phase 4: global pulse broadcast (unchanged behaviour).
        let pulse = runtime.next_pulse();
        let _ = deltas.send(pulse);
    }
}

fn apply_mutation_owned(
    runtime: &mut SimulationRuntime,
    mutation: crate::runtime_view::Mutation,
) {
    use crate::runtime_view::Mutation;
    match mutation {
        Mutation::ApplyCommand { command, reply } => {
            // apply_client_command is async — use a blocking inline executor.
            // tick_loop runs inside a tokio task; we can simply call .await via
            // tokio::runtime::Handle::current().block_on, OR convert the entire
            // tick_loop body to async (which it already is) and `.await` inline.
            let result = tokio::runtime::Handle::current().block_on(
                runtime.apply_client_command(command),
            );
            let _ = reply.send(result);
        }
        Mutation::SubscriptionDiff { added, removed, reply } => {
            runtime.apply_subscription_diff(added.iter(), removed.iter());
            let world_id = runtime.world_id_for_persist().clone();
            let tick = runtime.mobility_tick();
            let snapshots: Vec<abutown_protocol::MobilityChunkSnapshotDto> = added
                .iter()
                .map(|coord| {
                    let snapshot = runtime.mobility().build_chunk_snapshot(*coord);
                    chunk_snapshot_to_dto(&snapshot, runtime.mobility(), &world_id, tick)
                })
                .collect();
            let _ = reply.send(snapshots);
        }
        Mutation::MarkChunkSnapshotsPersisted { coords } => {
            runtime.mark_chunk_snapshots_persisted(&coords);
        }
    }
}
```
