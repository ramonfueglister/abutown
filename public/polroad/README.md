# PolRoad Vehicle Sprites

`polroad_private_cars.png` is an extracted runtime atlas built from the PolRoad 5.5 NewGRF road vehicle set.

Source package:
- Name: PolRoad 5.5
- BaNaNaS package ID: `4d430207`
- Content ID resolved through the OpenTTD content service: `6147473`
- Download: `https://bananas-cdn.openttd.org/newgrf/4d430207/c3aaab4fe3d3956af4d82ca62091cd5d/4d430207-PolRoad-5.5.tar.gz`

The atlas keeps only real road-vehicle sprite frames from the decoded GRF and packs them as 44 rows of 8 directional frames. It includes private passenger cars such as Ford T, CWS T-1, Opel Kapitan, FSO Warszawa, Polski Fiat 125p, FSO Polonez, Daewoo Lanos, Opel Astra J, Polski Fiat 508, and Polski Fiat 126p variants.

PolRoad is GPL v2. The original license text is included as `LICENSE.txt`.
